HowTo install and use jCalendar

Step 1 : Extract file from zip file

jCalendar is composed of one CSS file and one script file. These files are present in the archive file.

Step 2 : Include files in your project

Include the CSS File

Add this following code line in the head section :

<link rel="stylesheet" href="CSS/jCalendar.css" type="text/css" media="screen">

Include the Script File

Add this following code line in the head section :

<script type="text/javascript" src="Scripts/jCalendar.js"></script>

Step 3 : How to use in your HTML page

To create a new input that working with jCalendar class, you have to define the input type to "jCalendar".

<input type="jCalendar" />
